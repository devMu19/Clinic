# Week3 - Security And Delpoyment
## Getting started
- Fork the repo.
- Clone the forked repo into your machine.
- Create `.env`, you may copy it from `.env.exmaple` file and do your changes.
- `npm i`
- `npm run seed`.
- `npm run dev`.

## Envornment Variables file.
```
HOST=localhost
PORT=27017
APP_PORT=5000
DATABASE=clinic
JWT_SECRET_KEY="your_secret_key_here"
```
